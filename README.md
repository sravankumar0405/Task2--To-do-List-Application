# 📝 Tailwind CSS & Vanilla JavaScript To-Do List App

A sleek, responsive, and highly functional **To-Do List Application** built using **HTML5**, **Tailwind CSS v4 (via CDN)**, and optimized **Vanilla JavaScript**. This project focuses on a clean layout, interactive states, and inline task editing features.

---

## 🚀 Features

- ✨ **Add Tasks Easily**: Type a task and hit the **"ADD TASK"** button or simply press the `Enter` key.
- ✏️ **Inline Task Editing**: Modify existing tasks on the fly! Clicking "Edit" transforms the static task into an editable input field. Clicking "Save" locks your updates in place.
- ❌ **Instant Deletion**: Remove completed or unwanted tasks with a single click using the dedicated delete button.
- 🎨 **Modern Design**: Styled using a custom palette with vibrant violet, indigo, and zinc elements.
- 📱 **Fully Responsive Layout**: Built with a mobile-first mindset, adapting beautifully from small screens to desktop viewports.
- 🔤 **Custom Typography**: Utilizes Google Fonts integration (`Oswald`) to bring a professional, distinct look to the user interface.

---

## 🛠️ Tech Stack & Architecture

- **Frontend Structure**: HTML5 Semantic Markup
- **Styling Framework**: [Tailwind CSS v4](https://tailwindcss.com/) (loaded dynamically via the official `@tailwindcss/browser` package)
- **Custom Styling**: Core CSS fallback & font definitions (`style.css`)
- **Logic & Functionality**: Modern Vanilla JavaScript (DOM manipulation and Event Listeners)
